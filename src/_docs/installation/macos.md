## Installing Lintje with Homebrew

To install Lintje on macOS use the [Homebrew](https://brew.sh) tap.

```
brew tap lintje/lintje
brew install lintje
```

## Updating Lintje through Homebrew

Whenever a new version comes out run the following:

```
brew update
brew upgrade lintje
```
